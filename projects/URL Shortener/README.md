# URL Shortener using Tkinter


## Tech stack:
• Python


## Libraries used:
• Tkinter

• pyshorteners

• Messagebox

## To install external modules:
• Run pip install tkinter

• Run pip install pyshorteners


## To execute the project:
• Run Application main.py
